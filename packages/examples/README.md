# @ujl-framework/examples - Example Material and Test Data

This package contains example UJL content files (`.ujl.json`), theme files (`.ujlt.json`), and test data that can be used across the UJL Framework for development, testing, and documentation purposes.

## Structure

```
examples/
├── content/          # Example .ujl.json files
├── themes/           # Example .ujlt.json files
└── fixtures/         # Static assets and media
```
