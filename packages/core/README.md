# @ujl/core - The Heart of the UJL Framework

This package contains the core functionality of UJL: JSON schemas for `.ujl` and `.ujlt` files, a validator with AJV for schema validation, a renderer for **ContentFrame** generation (HTML/CSS/JS), TypeScript definitions for all UJL types, and utility functions for content management. It forms the foundation for all other UJL components and provides **AtomicComponents** (Text, Image, Video, Gallery, Maps, etc.) and **LayoutComponents** (Section, Grid, Hero, Carousel, etc.).
